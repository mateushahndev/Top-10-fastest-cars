# Top 10 Fastest Cars – Página Estática em HTML e CSS

Projeto pessoal desenvolvido de forma independente como primeiro projeto completo utilizando apenas HTML e CSS.

Inspirado na estética de sites antigos da década de 2010, o objetivo foi recriar a simplicidade estrutural e o estilo visual marcante daquela época.

## Contexto

Desenvolvido em janeiro de 2024, durante o primeiro mês de estudos em programação.

Este foi o primeiro projeto criado 100% de forma autoral, sem acompanhamento direto de aulas ou tutoriais.

A proposta surgiu da ideia de construir uma página temática sobre os carros mais rápidos do mundo, unindo interesse pessoal por automobilismo com prática técnica.

## Objetivos

- Praticar HTML semântico
- Trabalhar estruturação de conteúdo longo
- Aplicar CSS puro para estilização
- Organizar hierarquia visual (títulos, subtítulos, parágrafos)
- Treinar organização de arquivos

## Descrição

A página apresenta um ranking dos 10 carros mais rápidos do mundo, contendo:

- Estrutura completa com header, main e footer
- Uso de elementos como `figure`, `figcaption` e `address`
- Separação visual com divisores (`hr`)
- Estilização personalizada com tipografias externas (Google Fonts)
- Layout centralizado e organizado em blocos de conteúdo

## Tecnologias Utilizadas

- HTML5
- CSS3
- Google Fonts

## Observação

Projeto desenvolvido como hobby e prática pessoal, representando o início da jornada em desenvolvimento front-end.

Embora simples, marcou o primeiro exercício completo construído a partir de ideias próprias.
